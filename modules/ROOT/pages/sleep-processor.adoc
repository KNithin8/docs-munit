= Sleep Processor
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

The Sleep processor allows you to wait for the current thread for a specified time. This processor is normally used to simulate a delay in the processing of the flow:

[source,xml,linenums]
----
<munit-tools:sleep time="100" timeUnit="MILLISECONDS"/>
----

[%header%autowidth.spread]
|===
|Attribute Name |Description

|`time`
| The time period your test will wait for sleep.

|`timeUnit`
| Specifies the unit in which the `time` value is measured.

Possible values are:

* `MICROSECONDS`
* `MILLISECONDS`
* `MINUTES`
* `NANOSECONDS`
* `SECONDS`
* `HOURS`

If not specified, the default value is `MILLISECONDS`.
